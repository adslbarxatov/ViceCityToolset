## General information

This is a tool for adjusting weather parameters in the game. You can specify:
- Lighting for static and dynamic objects (color)
- Direct light effect (color)
- Sky top and bottom colors
- Water surface color
- Cloud colors
- Sun (core and corona) colors
- Sun size and brightness
- Shadows intensity
- Water opacity
- Maximum visible distance
- Fog effect

Any of these values can be specified for every weather type (sunny, cloudy, gray, foggy, rainy, windy)
and for every 24 hours in a day
