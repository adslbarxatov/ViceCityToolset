## General information

**Vehicles handling editor** – a tool for visual adjustment of characteristics
of cars, boats, bikes, planes and helicopters contained in the file `handling.cfg`.
Here you can set up:
- Vehicle mass and its centre
- Vehicle dimensions
- Traction settings
- Submerging percentage
- Number of gears
- Maximum velocity
- Drive and engine types
- Engine acceleration
- Brakes behavior
- Suspension values
- Seat placement
- Damage multiplier
- Vehicle price (for rating)
- Lights placement
- Boost flags
- Door types
- Exhaust flags
- Wheels flags
- Seat type

Special adjustments for bikes, planes, helicopters and boats are also available. Any parameter
can be separately set to every applicable vehicle type
