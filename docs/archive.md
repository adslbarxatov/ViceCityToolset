## General information

This is a tool for fast modifying of the main game archive (IMG / DIR pair).
It supports:
- File exctraction (selected item)
- File replacement with auto archive rebuilding (selected item)
- File edition – all-in-one procedure (selected item):
    - extracting file
    - opening for edition in the default app
    - waiting for completion
    - adding the file back to archive
    - rebuilding the archive
- File name filter
