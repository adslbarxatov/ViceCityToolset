## Общая информация

Это инструмент для быстрого редактирования основного архива игры (пара IMG / DIR).
Поддерживается:
- Извлечение файла (выбранный элемент)
- Замена файла с автоматической пересборкой архива (выбранный элемент)
- Редактирование файла – комплексная процедура (выбранный элемент):
    - извлечение файла
    - открытие для редактирования в приложении по умолчанию
    - ожидание завершения
    - добавление файла обратно в архив
    - пересборка архива
- Фильтр по имени файла
